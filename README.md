-> Student Login Portal
A responsive and modern Student Login Portal developed using HTML and CSS with a clean and user-friendly interface.

-> Features
* Responsive login page design
* Email and password input fields
* Modern gradient UI
* Flexbox-based layout
* Font Awesome icons
* Hover effects and smooth styling

-> Technologies Used

* HTML5
* CSS3
* Flexbox
* Font Awesome
